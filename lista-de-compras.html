Lista de Compras

Crie uma página para uma aplicação de lista de compras. A página deverá conter um campo de entrada e um botão para cadastrar os itens. Ao clicar no botão, o item deverá ser adicionado e deverá seguir uma sequência numérica de ordem de forma automática.

  • Adicionar itens dinamicamente: Digitar um item no campo de entrada e clicar no botão "Adicionar Item".
  • Testar a funcionalidade: Os itens devem ser adicionados em uma lista ordenada com numeração automática.
  • Validação: Garantir que itens vazios não possam ser adicionados.
  • Permitir que o item seja adicionado tanto ao clicar no botão, quanto ao pressionar "enter" após digitar o texto.
  • O campo de entrada deverá ser limpado sempre que um item for adicionado.
